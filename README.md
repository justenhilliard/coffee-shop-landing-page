# Cafe Twilight

Cafe Twilight is a simple cafe website made with HTML and CSS. It has a dark evening theme with a hero section, featured drinks, a "Why Cafe Twilight" section, and hours/location information.

## About the Project

This project is practice for building a basic website layout. The main goal is to make a cafe page that feels cozy, calm, and a little late-night themed.

## What's Included

- A homepage with a header and navigation
- A full-screen hero section
- Featured drink cards
- A scrolling "Why Cafe Twilight" section
- Hours and location section
- Custom styling with CSS

## Files

- `index.html` - the main webpage
- `styles.css` - all of the styling for the site

## How to View It

Open `index.html` in a browser.

You can also edit the page by changing the HTML in `index.html` and the design in `styles.css`.

## Notes

This website is still a work in progress. Some links may not go anywhere yet, and more sections can be added later like a full menu, gallery, reservation page, or working contact form.

## Project Goal

The goal of this project is to practice:

- HTML page structure
- CSS layout
- Navigation
- Cards and sections
- Background images
- Matching colors across a website
